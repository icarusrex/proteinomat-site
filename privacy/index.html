2. Dados que recolhemos

Recolhemos apenas os dados estritamente necessários para responder ao teu pedido ou prestar os serviços solicitados:

4. Com quem partilhamos os teus dados

Não vendemos nem cedemos os teus dados a terceiros para fins de marketing. Partilhamos dados apenas com:

  • Prestadores de serviços técnicos (alojamento do website, email, processamento de formulários), todos sob contrato e com garantias adequadas de proteção.
  • Autoridades públicas, apenas quando legalmente obrigatório.
  • Parceiros territoriais, apenas no que diz respeito a dados operacionais necessários para a gestão de contratos com ginásios no seu território.

Alguns dos nossos subprocessadores (ex: Formspree para formulários) podem transferir dados para fora do EEE; nestes casos, garantimos cláusulas contratuais padrão aprovadas pela Comissão Europeia.

5. Prazo de conservação

  • Contactos sem seguimento: 12 meses
  • Candidaturas de parceiros: 24 meses após resposta
  • Relações contratuais: duração do contrato + 10 anos (obrigações fiscais)
  • Dados de tráfego/analytics: 14 meses

6. Os teus direitos

Enquanto titular dos dados, tens o direito de:

  • Aceder aos dados que temos sobre ti
  • Pedir a retificação de dados incorretos
  • Pedir o apagamento ("direito ao esquecimento")
  • Limitar ou opor-te ao tratamento
  • Obter os teus dados num formato portável
  • Retirar o consentimento a qualquer momento (sem afetar tratamentos anteriores)
  • Apresentar reclamação à Comissão Nacional de Proteção de Dados (www.cnpd.pt)

Para exercer qualquer direito, escreve para aron@proteinomat.com. Respondemos em até 30 dias.

7. Segurança

Aplicamos medidas técnicas e organizativas razoáveis para proteger os teus dados: encriptação HTTPS em todo o site, acesso restrito a dados pessoais, backups encriptados, política interna de segurança, e revisão periódica de subprocessadores.

8. Alterações a esta política

Podemos atualizar esta política periodicamente. A data da última atualização está indicada no topo. Alterações materiais serão comunicadas no website e, quando aplicável, por email.

9. Contacto

Qualquer dúvida sobre esta política ou sobre o tratamento dos teus dados: aron@proteinomat.com.
